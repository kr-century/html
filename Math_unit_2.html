Introduction to the Unit

Number Theory studies properties of integers and is very important for Computer Science & AIML students because it is the foundation of:

  • Cryptography (RSA, blockchain, secure apps)
  • Hashing & checksums
  • Random number generation
  • Algorithm optimization & competitive programming

Total CI Hours: 12 | Mapped to CO-2 | Weightage in ETA: \~14 marks

4 Main Topics:

  1. Euclid’s Division Algorithm, GCD & LCM
  2. Prime Numbers & Fundamental Theorem of Arithmetic
  3. Congruences & Modular Arithmetic
  4. Chinese Remainder Theorem (CRT)

Topic 1: Euclid’s Division Algorithm, GCD & LCM

Key Concepts: Division algorithm, Euclidean algorithm, Extended Euclidean (for linear combination ax + by = gcd), LCM formula.

Important for: Fraction reduction, cryptography (modular inverse), programming contests.

15 Practice Questions:

  1. Find GCD(336, 192) using Euclid’s algorithm.
  2. Express GCD(561, 357) as 561x + 357y using extended Euclidean algorithm.
  3. Find LCM of 48, 72, and 108.
  4. Find integers x, y such that 119x + 85y = 17.
  5. Prove that GCD(a, b) = GCD(b, a mod b).
  6. Find GCD(1729, 1001) and LCM.
  7. Solve 455x + 273y = 91 using extended Euclidean.
  8. Reduce fraction 924/660 to lowest terms. Find GCD first.
  9. Find GCD(2⁵ × 3³ × 7, 2³ × 3² × 5² × 11) using prime factors.
  10. Find x, y such that 693x – 504y = 63.
  11. Prove LCM(a, b) × GCD(a, b) = |a × b|.
  12. Find GCD(4096, 1458) and express in linear form.
  13. Find smallest number divisible by 12, 18, 30 (use LCM).
  14. Use Euclidean algorithm to find GCD(987654, 123456).
  15. Find x, y for 1001x + 777y = 7.

  12. Find 2⁵⁰ mod 17 using modular exponentiation.
  13. Find inverse of 23 modulo 37.

Topic 4: Chinese Remainder Theorem (CRT)

Key Concepts: System of congruences with coprime moduli, unique solution mod M.

15 Practice Questions:

  1. Solve: x ≡ 2 (mod 3), x ≡ 3 (mod 5), x ≡ 1 (mod 7)
  2. Solve: x ≡ 4 (mod 5), x ≡ 3 (mod 7), x ≡ 2 (mod 11)
  3. Find x: x ≡ 1 (mod 4), x ≡ 5 (mod 9)
  4. Solve: x ≡ 7 (mod 11), x ≡ 4 (mod 13)
  5. Smallest number which when divided by 5 leaves 2, by 7 leaves 3, by 11 leaves 4.
  6. Prove CRT for two coprime moduli.
  7. How is CRT used in RSA decryption?
  8. Solve: x ≡ 3 (mod 8), x ≡ 5 (mod 13)
  9. Find x: x ≡ 2 (mod 7), x ≡ 4 (mod 9), x ≡ 1 (mod 5)
  10. Solve system with moduli 4, 6, 15 (check if solvable).
  11. Find general solution: x ≡ 0 (mod 5), x ≡ 1 (mod 8), x ≡ 4 (mod 9)
  12. Smallest positive x: x ≡ 5 (mod 6), x ≡ 4 (mod 35)
  13. Apply CRT to find day when three bells ring together (moduli given).
  14. Solve: x ≡ 1 (mod 3), x ≡ 2 (mod 4), x ≡ 3 (mod 5)
  15. Explain CRT application in computer science with example.
